Quick Colour Buttons for Gmail

One-click text colours for Gmail compose — Blue, Green, Red, Orange, Grey.

⬇️ Download the extension

Free • Works in Google Chrome & Edge • ~1 minute to install

Install in 3 steps

  1. Download & unzip. Click the button above, then right-click the downloaded file → Extract All. Remember where the gmail-quick-colour folder lands.
  2. Open the extensions page. In Chrome, go to chrome://extensions and turn on Developer mode (toggle, top-right corner).
  3. Load it. Click Load unpacked and pick the unzipped gmail-quick-colour folder. That's it! Open Gmail and start composing. ✅
⚠️ Keep the folder. Don't delete the unzipped folder — Chrome runs the extension from it. Put it somewhere permanent (e.g. Documents).

How to use

Select text in a Gmail compose window — a floating colour bar pops up. Click a colour, or use a shortcut while text is selected:

ColourShortcut
BlueCtrl+Shift+1
GreenCtrl+Shift+2
RedCtrl+Shift+3
OrangeCtrl+Shift+4
Greyset it once — see below
The Grey button works instantly. For its keyboard shortcut, Chrome needs a one-time setup: go to chrome://extensions/shortcuts → "Apply Grey Color" → press Ctrl+Shift+5.
